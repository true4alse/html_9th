a태그(링크태그)

사용자를 의도한 목적지로 보내주는 태그

a태그는 anchor라는 뜻으로 사용자를 원하는 목적지를 이동시켜줄 때 사용한다.

a태그는 href속성없이 사용되지 않는다. href속성에 들어가는 속성값은 목적지로 갈 수있는 경로를 입력하면 된다.

a태그는 마치 강조태그처럼 필요한 단어나 문장을 강조하는 것처럼 링크가 필요한 곳에 자유롭게 사용하면 된다.

a태그에서 target속성은 기본값은 _self로 설정되어있어서 사용자가 링크태그를 클릭했을 때 사용자가 보고있는 현재탭에서 이동된다. 이 값은 기본값이므로 target속성을 코드에 쓰지 않아도 a태그에 항시 적용되는 값이다.

target속성값을 _blank로 설정하면 기본적으로 설정되어있던 _self값은 무시되고 현재탭에서 열리지 않고 새탭에서 사용자가 클릭한 링크의 목적지 페이지가 열리게 된다.

경로입력 방법

절대경로/상대경로 2가지 방법이있다. 절대경로방법은 인터넷 주소를 복사해와서 그대로 사용하는 방법이며, 주소표시창에있는 주소를 그대로 복사해서 사용하면 된다.

상대경로

현재파일에서 목적지파일까지의 경로를 표현하는 방법

그래서 동일한 목적지 파일에 도달한다는 가정하에 선택된 현재파일의 위치에 따라서 경로는 달라질 수 있다

. -> 현재 파일의 위치를 불러온다

/ -> 폴더를 연다

.. -> 상위 폴더의 위치를 불러온다
